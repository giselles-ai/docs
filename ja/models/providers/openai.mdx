## GPT-5 シリーズモデル

OpenAI の最新かつ最も先進的なモデルファミリーである GPT-5 シリーズをご紹介します。これらのモデルは、強化された推論機能、より高速な処理速度、改善された効率性を特徴とし、幅広いタスクにわたってパフォーマンスの新しいベンチマークを設定しています。

### gpt-5

GPT-5 は OpenAI のフラッグシップモデルで、様々な領域におけるコーディング、複雑な推論、エージェンティックなタスクの新たな基準を設定しています。専門家レベルの知能と深い推論エンジンを内蔵し、多段階問題、技術文書作成、テキスト・コード・画像の分析において例外的な能力を発揮します。ウェブ検索機能もサポートしています。

*   **Context Window:** 400,000 tokens
*   **Max Output Tokens:** 128,000 tokens
*   **Knowledge Cutoff:** October 1, 2024
*   **Inputs:** Text, Image
*   **Availability:** Pro Plan

### gpt-5-mini

GPT-5 mini は、明確に定義されたタスクに最適化された、より高速でコスト効率に優れた GPT-5 のバージョンです。強力な推論能力を維持しながら、知能、速度、手頃な価格のバランスを取っています。画像入力とウェブ検索をサポートし、多くの一般的な用途において汎用的な選択肢となっています。

*   **Context Window:** 400,000 tokens
*   **Max Output Tokens:** 128,000 tokens
*   **Knowledge Cutoff:** May 31, 2024
*   **Inputs:** Text, Image
*   **Availability:** Free Plan

### gpt-5-nano

GPT-5 nano は GPT-5 シリーズで最も高速で費用対効果に優れたモデルです。要約や分類などの高スループットタスク用に設計され、迅速かつ効率的なパフォーマンスを提供します。テキストと画像の入力をサポートしていますが、ウェブ検索機能は含まれていません。

*   **Context Window:** 400,000 tokens
*   **Max Output Tokens:** 128,000 tokens
*   **Knowledge Cutoff:** May 31, 2024
*   **Inputs:** Text, Image
*   **Availability:** Free Plan

## 推論モデル

これらの o シリーズモデルは、推論を伴う複雑な多段階タスクに優れています。

### o4-mini
コーディングおよびビジュアルタスクにおいて効率的なパフォーマンスで効果的な推論に最適化された、より高速で手頃な価格の o シリーズ推論モデルです。速度、コスト、推論能力のバランスを提供します。画像入力をサポートし、o3 と同じ大きなコンテキストウィンドウと出力制限を共有しています。ウェブ検索もサポートしています。
*   **Context Window:** 200,000 tokens
*   **Max Output Tokens:** 100,000 tokens
*   **Knowledge Cutoff:** June 1, 2024
*   **Inputs:** Text, Image
*   **Availability:** Pro Plan

### o3
OpenAI の最も強力な推論モデルで、数学、科学、コーディング、視覚推論タスクにおいて高い基準を設定しています。技術文書作成、指示への従順性、多段階問題におけるテキスト・コード・画像の分析に優れています。画像入力をサポートし、大きなコンテキストウィンドウを持つため、細心の推論と安定性を要求する深い分析や複雑なワークフローに最適です。ウェブ検索もサポートしています。
*   **Context Window:** 200,000 tokens
*   **Max Output Tokens:** 100,000 tokens
*   **Knowledge Cutoff:** June 1, 2024
*   **Inputs:** Text, Image
*   **Availability:** Pro Plan

## フラッグシップモデル

幅広い複雑なタスクに適した、汎用性が高く知能に優れたモデルです。

### gpt-4.1
フラッグシップGPT-4.1モデルは、様々な領域における複雑なタスクと問題解決に優れています。大幅に改善されたコーディング能力、指示への従順性、そして約100万トークンの巨大なコンテキストウィンドウを特徴としています。テキストと画像の入力をサポートし、大きな文書やコードベースの深い分析に適しています。
*   **Context Window:** 1,047,576 tokens
*   **Max Output Tokens:** 32,768 tokens
*   **Knowledge Cutoff:** June 1, 2024
*   **Inputs:** Text, Image
*   **Availability:** Pro Plan

### gpt-4o
汎用的なGPT-4o モデル（「o」は「omni」の意）は、高度なテキスト生成、マルチモーダル画像入力、統合ウェブ検索（ツールとして利用可能）を含む包括的な機能を提供します。構造化出力と関数呼び出しをサポートしています。128kトークンのコンテキストウィンドウにより、複雑な分析タスク、マルチモーダル理解、最新情報が必要な汎用高度アプリケーションに最適です。
*   **Context Window:** 128,000 tokens
*   **Max Output Tokens:** 16,384 tokens
*   **Knowledge Cutoff:** October 1, 2023
*   **Inputs:** Text, Image
*   **Availability:** Pro Plan

## 画像生成モデル

これらのモデルは、テキストや画像入力から高品質な画像を生成することに特化しています。

### gpt-image-1
OpenAIの最先端画像生成モデルです。テキストと画像の両方の入力を受け入れ、画像出力を生成するネイティブマルチモーダル言語モデルです。このモデルは異なる品質レベル（Low、Medium、High）を提供し、様々な画像サイズをサポートしており、使用例の要件に基づいた柔軟な生成が可能です。
*   **Pricing:** Input text: $5.00 per 1M tokens, Input images: $10.00 per 1M tokens, Output images: $40.00 per 1M tokens
*   **Quality Options:** Low, Medium, High
*   **Supported Dimensions:** 1024x1024, 1024x1536, 1536x1024
*   **Knowledge Cutoff:** April 2025 (estimate based on release date)
*   **Inputs:** Text, Image
*   **Outputs:** Image
*   **Availability:** Pro Plan

## Giselle でのプラクティス

Proユーザーにはまず **gpt-5** を Giselle における汎用性の高い主力モデルとして推奨いたします。複雑なコーディング、ビジネス文書作成、詳細な分析、高度な研究など、様々なタスクにおいて、能力、知能、機能（ツールによるウェブ検索を含む）の比類のないバランスを提供します。GPT-5 は高い信頼性と精度を持つよう設計されており、幻覚を大幅に削減し、指示への従順性を向上させています。

絶対的に最高レベルの推論を要求するタスクや非常に大きなコンテキスト（最大200kトークン）を扱う場合は、**o3** または **o4-mini** を検討してください。どちらもウェブ検索をサポートし続けており、特定の研究や最新情報の取得ニーズに対して汎用性があります。**o3** は深さと安定性に優れた選択肢であり、**o4-mini** はそれぞれのコンテキストウィンドウに対してより優れたコスト効率で堅実な推論を提供します。

**GPT-4.1シリーズ**（gpt-4.1、gpt-4.1-mini、gpt-4.1-nano）は、全ての階層において巨大な **約100万トークンのコンテキストウィンドウ** を提供し続け、改善されたコーディングと指示への従順性を備えており、GPT-5の400kコンテキストでは不十分な極めて大きな文書やコードベースの処理に理想的です。
*   最大のコンテキストを要求する最も厳しいタスクには **gpt-4.1** を使用してください。
*   **gpt-4.1-mini** は大きなコンテキストタスクに対して低コストでバランスを提供します。
*   **gpt-4.1-nano** は100万トークンのコンテキストウィンドウを活用する最も経済的な選択肢です。

Freeプランのユーザーや、中程度に複雑なタスクでコストと速度を優先する方には、**gpt-5-mini** と **gpt-5-nano** が新たに推奨される選択肢であり、GPT-5シリーズの能力内で強力なパフォーマンスと効率性を提供します。

画像生成のニーズに対しては、**gpt-image-1** が高品質な結果を提供し、テキストと画像の両方の入力をサポートしています。このモデルは特定の要件に基づいてコストと詳細のバランスを取るために、異なる品質階層を提供しています。

ワークフローでこれらのモデルを組み合わせることで、それぞれの特定の強みを活用できます。例えば、高度な推論とコーディングには `gpt-5` を使用し、コスト効率的なタスクには `gpt-5-mini` を使用します。GPT-5のコンテキストを超える極めて大きな文書の分析を含むタスクの場合、`gpt-4.1` またはそのコスト最適化バリアントは優れた選択肢であり続けます。

詳細な仕様とOpenAIが直接提供するモデルの全範囲については、[公式OpenAIドキュメント](https://platform.openai.com/docs/models)をご確認ください。
