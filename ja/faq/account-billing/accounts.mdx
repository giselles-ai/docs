## Giselle アカウントを作成するには？

以下の方法で Giselle アカウントを作成できます：

- **メールとパスワード**: [studio.giselles.ai/signup](https://studio.giselles.ai/signup) でメールアドレスとパスワードを使用してサインアップ
- **Google**: Google アカウントで直接サインアップ
- **GitHub**: GitHub アカウントで直接サインアップ

メールでサインアップした場合、アカウントを確認するための認証メールが届きます。

## アカウントにログインするには？

[studio.giselles.ai/login](https://studio.giselles.ai/login) にアクセスし、以下の方法でログインできます：

- メールとパスワード
- Google アカウント
- GitHub アカウント

## 複数のログイン方法をアカウントに接続できますか？

はい。[アカウント設定 > 認証](https://studio.giselles.ai/settings/account/authentication) から Google や GitHub アカウントを Giselle アカウントに接続できます。これにより、接続した任意の方法でログインできるようになります。

## ログイン方法を解除するには？

[アカウント設定 > 認証](https://studio.giselles.ai/settings/account/authentication) に移動し、解除したいプロバイダーの横にある「Disconnect」をクリックします。少なくとも1つのログイン方法を維持する必要があります。

## プロフィール情報を更新するには？

[アカウント設定 > 一般](https://studio.giselles.ai/settings/account/general) で以下を更新できます：

- 表示名の更新
- プロフィール画像のアップロードまたは変更（JPG、PNG、GIF、WebP に対応）

## アカウントを削除するには？

アカウントを削除するには、サポートチーム（[support@giselles.ai](mailto:support@giselles.ai)）にお問い合わせください。

## チームとは？

チームは、同僚と AI アプリを共同で作業できる共有 Workspace です。各ユーザーは複数のチームを作成し、所属することができます。

## 新しいチームを作成するには？

[アカウント設定](https://studio.giselles.ai/settings/account) に移動し、「Create New Team」をクリックします。チーム名を入力して開始してください。

## チームメンバーの役割は？

- **Admin**: チーム設定、請求、チームメンバーを管理可能
- **Member**: チームリソースを使用できるが、設定、請求、メンバーの管理は不可

## チームメンバーを招待するには？

チームの Admin は [チーム設定 > メンバー](https://studio.giselles.ai/settings/team/members) から新しいメンバーを招待できます。招待したい人のメールアドレスを入力すると、招待メールが送信されます。チームメンバーの追加には Pro プランが必要です。

## チームメンバーを削除するには？

チームの Admin は [チーム設定 > メンバー](https://studio.giselles.ai/settings/team/members) からメンバーを削除できます。削除したいメンバーの横にあるメニューをクリックし、「Remove」を選択します。

## チームを脱退できますか？

はい、少なくとも1つの他のチームに所属している場合は脱退できます。唯一のチームの場合は、まず別のチームを作成するか参加する必要があります。

## チームを削除するには？

Free プランのチームのみ削除できます。[チーム設定 > 一般](https://studio.giselles.ai/settings/team/general) に移動し、削除オプションを探してください。削除する前に、少なくとも1つの他のチームが必要です。
