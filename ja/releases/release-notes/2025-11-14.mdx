このアップデートでは、組織向けEnterpriseプラン、GitHub Issues Vector Store、モバイル対応ナビゲーションを提供します。

## Enterpriseプラン

高度なニーズを持つ組織向けに**Enterpriseプラン**を導入しました：

- AIモデル呼び出しの利用枠の増加
- ワークスペースバリデーションを含むチームベースのアクセス制御
- 請求およびサブスクリプション管理の強化

Enterprise機能の詳細についてはお問い合わせください。

## GitHub Issues Vector Store

**GitHub Issues Vector Store**がデフォルトで利用可能になりました：

- GitHubリポジトリのIssueを検索可能なナレッジベースとしてインデックス化
- AIエージェントがIssueのディスカッション、バグレポート、機能リクエストを参照可能に
- プロジェクト履歴を理解したコンテキスト認識型ワークフローを構築

## モバイルナビゲーション

小さな画面でのナビゲーションを改善する**モバイルドロワーメニュー**を追加しました。どこからでも、どのデバイスからでも、すべてのワークスペースと設定にアクセスできます。

## UI改善

- 新しいデザインのメールテンプレート
- グラデーションとシャドウを使用したノードスタイリングの強化
- ワークスペース一覧に実行回数を表示
- リポジトリ名やファイル名が切り詰められた際のツールチップ表示
- Document Vector Storeインターフェースの改善

## バグ修正

- ファイルアップロードノードのコピー動作を修正
- Document Vector Storeでファイルセレクターが二重に開く問題を修正
- チームドロップダウンメニューのナビゲーション問題を修正
- ログアウトメニュー項目のクリック領域を改善
