このアップデートでは、テキスト生成へのPostgreSQLツールサポート、GitHub Vector Storeのデフォルト有効化、画像のクリップボードペースト、ワークフロー実行ボタンを導入します。

## PostgreSQLツールサポート

テキスト生成ノードに**PostgreSQLツールサポート**を追加しました。PostgreSQLデータベースに接続し、AIエージェントがワークフロー内で直接データをクエリ・分析できます。

## GitHub Vector Store

**GitHub Vector Store**がすべてのユーザーにデフォルトで有効になりました。GitHubリポジトリをインデックス化し、AIによるコード支援やドキュメント作成のナレッジベースとして使用できます。

## 画像のクリップボードペースト

Image File Nodeに**クリップボードから直接画像をペースト**できるようになりました。画像をコピーしてワークフローにペーストするだけで、より速く画像を扱えます。

## ワークフロー実行ボタン

ワークフロー実行用の新しい**実行ボタン**。ワークフローデザイナーから直接ワンクリックでワークフローを開始できます。

## チームメンバー管理

チームメンバーが自分で**チームを離脱**できるようになり、ユーザーがチームメンバーシップをより自由に管理できます。

## 差分インジェスト

GitHubリポジトリの**バージョン追跡付き差分インジェスト**を実装しました。変更されたファイルのみが再インデックス化され、リポジトリの更新がより高速で効率的になりました。
