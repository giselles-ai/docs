このリリースでは、GPT-5.2 Codex モデルの追加、Playground と Tasks の体験改善、GitHub Trigger の実行中コメント制御を提供します。

## GPT-5.2 Codex モデル

OpenAI のコード特化モデル **GPT-5.2 Codex** のサポートを追加しました。ソフトウェアエンジニアリングタスクに最適化されており、Generator ノードで GPT-5.2 Codex を選択してコード生成や技術的なワークフローに活用できます。

## Playground アプリ選択

ワークフローデザイナーから直接アプリを選択し、Playground にジャンプして実行できるようになりました。アプリの構築からテストまでのワークフローがよりスムーズになります。

## Tasks ページの改善

- **入力値の展開表示** — タスクヘッダーで文字列入力が展開可能な形式で表示されるようになり、タスクページを離れずに長い入力を確認しやすくなりました

## GitHub Trigger の実行中コメント制御

GitHub イベントによるワークフロー実行時に、GitHub に「実行中」コメントを投稿するかどうかを制御できるようになりました。GitHub 連携ワークフローの通知動作をより柔軟にカスタマイズできます。

## バグ修正と改善

- アバターアップロードにマジックバイトバリデーションを追加し、無効なファイルタイプを防止
- GitHub Vector Store のサーバーサイドリポジトリアクセスバリデーションを追加
- アプリ切り替え時にタスク入力プレースホルダーが更新されない問題を修正
- Node.js を 22 から 24 にアップグレード
