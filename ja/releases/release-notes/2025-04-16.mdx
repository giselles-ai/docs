このアップデートでは、OpenAI Web Search連携、チーム招待システム、GitHub連携機能の強化を導入します。

## OpenAI Web Search Tool

テキスト生成に新しい**OpenAI Web Search Tool**を統合し、AIが応答生成中にリアルタイムでウェブ検索できるようになりました。

## OpenAI o3およびo4-miniモデル

高度な推論機能を持つ**OpenAI o3**および**o4-mini**モデルのサポートを追加。Claude 3.7 Sonnetに問題解決を強化する推論オプションも追加されました。

## チーム招待

コラボレーションのための完全な**チーム招待システム**：

- **チームメンバーの招待**: チームへの参加招待を送信
- **招待による参加**: 招待を受け入れてチームに参加
- **重複防止**: システムが重複した招待を防止

## GitHub連携の強化

- **GitHubアプリインストールフロー**: GitHub連携のセットアップを簡素化
- **追加GitHubツール**: GitHub操作の機能を拡張
- **Pull Requestイベント**: PR opened、ready_for_review、closedのワークフロートリガー
- **Issueイベント**: issues openedとclosedのワークフロートリガー

## Proモデルアクセス

Pro言語モデル用の新しい**アップグレードダイアログ**。明確なアップグレードパスでプレミアムAI機能に簡単にアクセスできます。

## Geminiアップデート

- **Gemini 2.5 Flash Preview**（04-17）モデルを追加
- **Gemini 2.5 Pro Preview**（03-25）モデルを追加
