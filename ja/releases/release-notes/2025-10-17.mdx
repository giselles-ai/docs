このアップデートでは、GitHub Discussionサポート、ノードのリデザイン、プラットフォーム全体の各種UI改善を提供します。

## GitHub Discussionトリガーとアクション

**GitHub Discussion**からワークフローをトリガーし、アクションを実行できるようになりました：

- リポジトリの新しいディスカッションに自動的に応答
- AIを活用したコミュニティサポートワークフローを構築
- ディスカッションのサマリーやインサイトを作成

## GitHub Triggerのカスタマイズ

GitHub Triggerで、設定後に**コールサインとラベル**をカスタマイズできるようになりました。最初から再設定することなく、トリガーの表示方法や識別方法を変更できます。

## ノードデザインの改善

複数のノードタイプのデザインを改善しました：

- **画像生成ノード** — 統一されたヘッダーと改善された画像表示でリデザイン
- **ファイルアップロードノード** — より良いファイル管理のためのレイアウト改善
- **クエリノード** — タブをスクロール可能なレイアウトに置き換え、ナビゲーションを容易に

## GitHub Tools設定

GitHub Tools設定パネルを改善しました：

- GitHub連携を設定するためのよりクリーンなインターフェース
- 設定オプションの整理を改善
- 視覚的な一貫性の向上

## バグ修正

- 設定でのモーダルダイアログレイアウトの問題を修正
- ワークフロービルドでの重複接続の処理を修正
- Workspacesヘッダーのハイライトを修正
