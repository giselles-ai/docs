このメジャーアップデートでは、完全にリデザインされたインターフェースを備えたV2ワークフローデザイナー、RAG/ベクトル検索用のGitHub Pull Requestローディング、多数のUI改善と内部アーキテクチャの更新を導入します。

## V2ワークフローデザイナー

レガシーUIを**V2ワークフローデザイナー**に置き換え、以下の機能を提供：

- 完全にリデザインされたワークフローインターフェース
- 改善されたノードの整理とグルーピング
- 簡素化されたプロパティパネルUI
- アイコンサポートを備えた再設計されたGitHub Actionプロパティパネル
- 新しいドロップダウンメニューコンポーネント

## GitHub Pull Requestローダー

RAG/ベクトル検索用の**GitHub Pull Requestローダー**を追加しました。リポジトリからPull Requestを読み込んでインデックス化し、AI搭載のコードレビューやドキュメントワークフローを実現できます。

## PostgreSQLツールの強化

PostgreSQLツールで**接続文字列のバリデーション**が利用可能になり、クエリを実行する前に設定エラーを検出できます。

## チームベースのフロー選択

ステージページに新しい**チームベースのフロー選択**コンポーネントを追加し、チーム間でワークフローを整理してアクセスしやすくなりました。

## 内部改善

- ステップ失敗後の早期リターンによるエラーハンドリングの改善
- 失敗後の残りのシーケンスをスキップし、アクトステータスを適切に更新
- アクトナビゲーションUIの簡素化とステータス用語の統一
- 可視性制御を備えた実行ボタンロジックの改善

## バグ修正

- スタジオの不整合なカラム名を修正
- インポートの問題を修正し、テストを更新
- 実行可能なアイテムがない場合に実行ボタンを非表示
