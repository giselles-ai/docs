このアップデートでは、Vector Store設定UI、新しいGitHub Triggerコンポーネント、OpenAI o3およびo4-miniモデルのWeb Searchサポートを導入します。

## Vector Store設定

ナレッジベースを管理するための新しい**Vector Store設定**UI。設定パネルから直接ベクトルストアを設定・整理できます。

## GitHub Trigger UI

イベント選択ワークフローを備えた**GitHub Trigger UIコンポーネント**を追加しました。イベントの選択やトリガー動作の設定を直感的なインターフェースで行えます。

## OpenAI o3 & o4-miniのWeb Search

OpenAI **o3**および**o4-mini**モデルで**Web Search**が有効になり、生成中にリアルタイムの情報をウェブ検索できるようになりました。

## ドロップダウンメニューの強化

**DropdownMenuコンポーネント**がグループ機能をサポートし、メニューアイテムを論理的なカテゴリに整理しやすくなりました。
