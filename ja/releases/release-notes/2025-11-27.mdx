このアップデートでは、Claude Opus 4.5のサポート、コンテンツ生成機能の大幅な強化、サブスクリプション管理機能の改善を提供します。

## Claude Opus 4.5が利用可能に

Anthropicの最も高性能なモデル**Claude Opus 4.5**のサポートを追加しました。Claude Opus 4.5は、複雑な推論タスク、クリエイティブライティング、繊細な分析に優れています。Generatorノードで Claude Opus 4.5を選択して、その高度な機能を活用できます。

## コンテンツ生成機能の強化

コンテンツ生成に関していくつかの改善を行いました：

- **ストリーミングUI** — AIが生成するコンテンツをリアルタイムで確認できます
- **クリップボードにコピー** — 生成されたコンテンツをワンクリックで簡単にコピー
- **ツール統合** — コンテンツ生成ノードでGitHubアクションなどのツールを直接使用可能に
- **ツール結果の表示改善** — ツールの実行結果を明確で整理された形式で確認できます

## サブスクリプション管理

- サブスクリプションのキャンセルUIを追加し、プラン管理が容易に
- 請求ポータルのサブスクリプションサポートを改善

## バグ修正と改善

- ソースノード削除時に孤立した入力が残る問題を修正
- GitHub Appインストールのエラーハンドリングを改善
- ワークフローデザイナーのコネクターアニメーションを強化
- 各種ツールバーとレイアウトの調整
