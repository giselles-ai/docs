## 利用可能なプラン

当社のプランおよび価格オプションの最新情報については、[価格ページ](https://giselles.ai/pricing) をご覧ください。

### Free プラン
AI エージェントを探求する個人開発者に最適です。

- 月あたり **30分間** の AI モデル使用（月次クレジット$3相当）
- 基本的な AI モデルへのアクセス
- ノーコードビルダーでのエージェント作成・デプロイ
- 単一ユーザーアカウント
- マネージドクラウドデプロイ

### Pro プラン
個人のパワーユーザー向けに設計されています。

- **月額$20**
- **無制限** の AI モデル使用（期間限定プロモーションオファー）
- すべてのプレミアム AI モデルへのアクセス
- Free プランのすべての機能を含む
- Document & GitHub Vector Stores
- 単一ユーザーアカウント
- マネージドクラウドデプロイ
- メールサポート

### Team プラン
AI エージェントでコラボレーションするチーム向けに設計されています。（近日提供予定）

- 料金については営業チームにお問い合わせください
- Pro プランのすべての機能を含む
- 複数ユーザーでのチームコラボレーション
- メンバー招待と役割管理
- 高度なチーム管理機能
- 優先メールサポート

## 月次 AI モデル使用量の詳細

各 Giselle サブスクリプションには、月次 AI モデル使用クレジットが含まれます：

- **Free プラン:** AI 使用の月次制限は30分間です。月次制限に達すると、次の請求サイクルまで AI 使用は一時停止されます。
- **Pro プラン:** 月額$20相当の AI 使用クレジットが含まれます。現在の期間限定プロモーション中は、標準月次クレジットを超えるすべての使用量が追加料金なしで無制限となります。

AI モデル使用クレジットは毎月更新され、未使用クレジットの繰り越しはありません。

## Vector Store クォータ

各プランには、Vector Stores の作成・登録数に制限があります。

### プランごとの制限

| プラン | Document Vector Stores | GitHub Repositories |
|--------|----------------------|-------------------|
| Free   | 利用不可               | 利用不可            |
| Pro    | 最大5個               | 最大3個            |
| Team   | 最大20個              | 最大10個           |

- **Document Vector Stores**: PDF、TXT、Markdown ファイルをアップロードして、AI ワークフローで活用できるナレッジベースを構築できます
- **GitHub Repositories**: GitHub リポジトリのコード、Issues、Pull Request を取り込んで検索可能にできます

<Note>
Vector Stores の詳細については、[Vector Stores 設定](/ja/guides/settings/team/vector-stores)をご覧ください。
</Note>

### 制限に達した場合の対処方法

#### Free プラン
- Vector Stores を使用するには、Pro または Team プランへのアップグレードが必要です

#### Pro プラン
- **Document Vector Stores**: 5個の制限に達した場合、既存の Vector Store を削除するか、Team プランへのアップグレードを検討してください
- **GitHub Repositories**: 3個の制限に達した場合、既存のリポジトリを削除するか、Team プランへのアップグレードを検討してください

#### Team プラン
- **Document Vector Stores**: 20個の制限に達した場合、既存の Vector Store を削除してください
- **GitHub Repositories**: 10個の制限に達した場合、既存のリポジトリを削除してください

## プラン変更および請求情報

### Free から Pro へのアップグレード
- アップグレード時、Free プランの未使用クレジットは破棄されます。
- 無制限プロモーション使用を含む Pro プランの特典は、アップグレード時に即座に利用可能になります。

### Pro から Free へのダウングレード
- ダウングレードは現在の請求サイクルの終了時に有効になります。
- Pro の特典はダウングレードが有効になるまで利用可能です。

### チームメンバーシップ
- **Free プラン:** 単一ユーザーに限定され、1つの Free チームのみの所属となります。
- **Pro プラン:** 単一ユーザーアカウントです。複数のチームメンバーシップが可能ですが、各チームは1ユーザーに限定されます。
- **Team プラン:** 複数のチームメンバーシップと無制限のチームサイズが可能です。料金詳細については営業チームにお問い合わせください。

### 追加使用料金（Pro プラン）
- 標準（非プロモーション）期間中、月額$20使用クレジットを超える Pro プランユーザーには、合理的な追加料金（10%手数料）が発生し、次のサブスクリプション支払いサイクルで自動的に請求されます。

### プロモーション期間
- Pro プランの現在の無制限使用オファーは一時的なものです。このプロモーション終了の1か月前にユーザーに通知されます。
