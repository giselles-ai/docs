This update introduces the Image Generation Node, GitHub webhook integration for workflow automation, Gemini Search Grounding, and a completely new Playground UI.

## Image Generation Node

New **Image Generation Node** for creating AI-generated images directly in your workflows. Configure image size, style, and other parameters to generate visuals on demand.

## GitHub Webhook Integration

Powerful **GitHub webhook integration** for automated workflows:

- **Workflow execution via webhooks**: Trigger workflows automatically from GitHub events
- **GitHub data injection**: Access GitHub context data (diffs, PR details) in your workflows
- **Issue and PR events**: Respond to repository activity automatically

## Gemini Search Grounding

**Gemini 2.0 Search Grounding** support allows AI to search and cite real-time information, providing more accurate and up-to-date responses with source references.

## New Playground UI

Completely redesigned **Playground interface** with:

- **Builder/Preview toggle**: Switch between building and testing modes
- **Improved canvas gestures**: Better trackpad support for navigation
- **Sample workspace creation**: Get started quickly with pre-built examples

## File Nodes

New file handling capabilities:

- **Image File Node**: Upload and use images in workflows
- **Text File Node**: Upload and process text files

## Language Model Access Control

**Subscription-based model access** with clear tier distinctions. Pro models now require appropriate subscription levels, with upgrade dialogs guiding users to the right plan.

## Settings Redesign

Complete overhaul of **settings pages**:

- Team settings and member management
- Billing and usage tracking
- Account and authentication settings
- Integration configuration
