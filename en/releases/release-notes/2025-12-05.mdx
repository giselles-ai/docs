This update brings official vector stores accessible to everyone, enhanced team navigation, and several quality-of-life improvements.

## Official Vector Stores Now Available to All Users

We're excited to announce that **official vector stores** are now accessible to all Giselle users:

- **Official Document Vector Stores** — Pre-built document knowledge bases that you can use in your workflows without any setup
- **Official GitHub Vector Stores** — Ready-to-use GitHub repository knowledge bases available as read-only resources

These official vector stores allow you to quickly leverage curated knowledge in your AI workflows, making it easier to get started with powerful context-aware AI applications.

## Improved Team Navigation

- Added team avatar icons to the workspaces list for easier team identification
- Team links now navigate directly to your workspaces
- Fixed an issue where the "Create team" button in the header was not working

## GitHub Integration Improvements

- Improved detection and error handling for GitHub integrations
- Added reconnection UI for expired GitHub credentials — you'll now see a clear prompt to reconnect when needed
- Vector store setup links now open in a new tab for a smoother workflow

## Bug Fixes

- Fixed GitHub tools configuration issues
- Resolved display issues with vector store status cards
- Improved settings visibility when no repository is selected
