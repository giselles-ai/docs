This major update introduces a completely redesigned workflow editor, Google Gemini model support, Google OAuth login, and comprehensive team management features.

## Redesigned Workflow Editor

Completely redesigned **workflow editor** with powerful new capabilities:

- **Graph-based workflow editor**: Visual node-based design for building AI workflows
- **Real-time execution tracking**: Monitor workflow progress as it runs
- **Keyboard shortcuts**: Quick actions for common operations
- **Versioned graph persistence**: Save and manage workflow versions
- **Flow retry capability**: Retry failed executions with state preservation
- **Agent duplication**: Copy agents with all settings and files

## Google Gemini Support

Added **Google Gemini** model support including **Gemini 2.0 Flash Exp**, expanding your AI model options for text generation.

## Google OAuth Login

New **Google OAuth** login option for easier account access alongside existing GitHub authentication.

## Team Management

Comprehensive **team management** system:

- **Team creation and settings**: Create and configure teams
- **Member management**: Add, remove, and manage team members with roles
- **Activity logs**: View detailed agent activity history
- **Usage tracking**: Monitor team usage and limits

## Multi-File Upload

**Multi-file upload** support with processing capabilities, allowing you to work with multiple documents in your workflows.

## Agent Improvements

- **Customizable system prompts**: Configure system prompts for text generation
- **Agent name management**: Easily rename and organize agents
- **Loading states**: Visual feedback during operations
