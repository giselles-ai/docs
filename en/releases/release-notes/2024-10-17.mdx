**Giselle is now open source!**

We're excited to announce that Giselle has been released as an open source project. This marks the beginning of our journey to build AI workflow automation tools together with the community.

## Open Source Launch

Giselle is now available on GitHub under an open source license. We believe in transparency and community-driven development, and we're inviting developers worldwide to contribute, explore, and build with us.

## What's Included

- **Full source code**: Complete access to the Giselle codebase
- **Documentation**: Contributing guidelines, code of conduct, and security policy
- **Community infrastructure**: Issue templates, pull request workflows, and code review processes

## Get Involved

We welcome contributions of all kinds:

- **Feature development**: Help build new capabilities
- **Bug fixes**: Identify and resolve issues
- **Documentation**: Improve guides and references
- **Feedback**: Share your ideas and use cases

Visit our [GitHub repository](https://github.com/giselles-ai/giselle) to get started.
