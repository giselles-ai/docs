This update introduces an Enterprise plan for organizations, GitHub Issues Vector Store, and mobile-friendly navigation.

## Enterprise Plan

Introducing the **Enterprise plan** for organizations with advanced needs:

- Higher usage quotas for AI model calls
- Team-based access control with workspace validation
- Enhanced billing and subscription management

Contact us to learn more about Enterprise features.

## GitHub Issues Vector Store

The **GitHub Issues Vector Store** is now available by default. You can now:

- Index your GitHub repository issues as a searchable knowledge base
- Let your AI agents reference issue discussions, bug reports, and feature requests
- Build context-aware workflows that understand your project history

## Mobile Navigation

We've added a **mobile drawer menu** for better navigation on smaller screens. Access all your workspaces and settings from anywhere, on any device.

## UI Improvements

- New email templates with refreshed design
- Enhanced node styling with gradients and shadows
- Workspace list now shows execution counts
- Tooltips for truncated repository and file names
- Improved Document Vector Store interface

## Bug Fixes

- Fixed file upload node copy behavior
- Fixed double file selector trigger in document vector store
- Fixed team dropdown menu navigation issues
- Improved log out menu item click area
