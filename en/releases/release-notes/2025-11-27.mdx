This update introduces Claude Opus 4.5 support, significant enhancements to content generation capabilities, and improved subscription management features.

## Claude Opus 4.5 Now Available

We've added support for **Claude Opus 4.5**, Anthropic's most capable model. Claude Opus 4.5 excels at complex reasoning tasks, creative writing, and nuanced analysis. You can now select Claude Opus 4.5 in your Generator nodes to leverage its advanced capabilities.

## Enhanced Content Generation

We've made several improvements to content generation:

- **Streaming UI** — Watch your AI-generated content appear in real-time as it's being created
- **Copy to clipboard** — Easily copy generated content with a single click
- **Tool integrations** — Content generation nodes can now use tools like GitHub actions directly
- **Improved tool results display** — See tool execution results in a clear, organized format

## Subscription Management

- Added subscription cancellation UI for easier plan management
- Improved billing portal with better subscription support

## Bug Fixes & Improvements

- Fixed orphaned inputs when source nodes are deleted
- Improved GitHub App installation error handling
- Enhanced connector animations in the workflow designer
- Various toolbar and layout adjustments
