This update introduces GPT-5 model series and Claude Opus 4.1 support, adds Anthropic's web search tool, enhances keyboard shortcuts in the workflow designer, and provides embedding profile selection for better vector store management.

## GPT-5 Model Series

Added support for the **GPT-5 model series** from OpenAI, bringing the latest reasoning capabilities to your workflows. This includes structured GPT-5 models optimized for various use cases.

## Claude Opus 4.1 Support

Added **Claude Opus 4.1** (claude-opus-4-1-20250805), Anthropic's powerful model for complex reasoning and creative tasks.

## Anthropic Web Search Tool

New **Anthropic Web Search** tool integration allows your AI agents to search the web and incorporate real-time information into their responses.

## Enhanced Keyboard Shortcuts

Implemented **robust keyboard shortcuts** with focus management for the workflow designer. Navigate and edit your workflows more efficiently with keyboard commands.

## Embedding Profile Selection

- Select embedding profiles in **VectorStoreNode** for customized search behavior
- New **embedding profile selection UI** in team settings
- Multi-embedding backend support for improved search accuracy

## Team & User Features

- **Team avatar registration** with header dropdown integration
- Display avatar images on team members page
- Create multiple sample workspaces for onboarding

## Download Functionality

Implemented **download functionality** for desktop and mobile act details, allowing you to export your workflow execution results.

## Bug Fixes

- Fixed staged triggers appearing after app duplication
- Fixed workflow not running with only a Generator Node
- Ensured avatars always render as perfect circles
- Fixed unique ID generation for sample workspace elements
