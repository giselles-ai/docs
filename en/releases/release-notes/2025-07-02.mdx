This update introduces Vector Store settings UI, new GitHub Trigger components, and Web Search support for OpenAI o3 and o4-mini models.

## Vector Store Settings

New **Vector Store settings** UI for managing your knowledge bases. Configure and organize your vector stores directly from the settings panel.

## GitHub Trigger UI

Added **GitHub Trigger UI components** with event selection workflow. Set up GitHub-based automation triggers with an intuitive interface for selecting events and configuring trigger behavior.

## OpenAI Web Search for o3 & o4-mini

**Web Search** is now enabled for OpenAI **o3** and **o4-mini** models, allowing these models to search the web for real-time information during generation.

## Enhanced Dropdown Menu

The **DropdownMenu component** now supports group functionality, making it easier to organize menu items into logical categories.
