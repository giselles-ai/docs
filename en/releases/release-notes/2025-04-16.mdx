This update introduces OpenAI Web Search integration, team invitation system, and enhanced GitHub integration features.

## OpenAI Web Search Tool

New **OpenAI Web Search Tool** integration for text generation, allowing AI to search the web in real-time while generating responses.

## OpenAI o3 and o4-mini Models

Added support for **OpenAI o3** and **o4-mini** models with advanced reasoning capabilities. Claude 3.7 Sonnet now includes a reasoning option for enhanced problem-solving.

## Team Invitations

Complete **team invitation system** for collaboration:

- **Invite team members**: Send invitations to join your team
- **Join via invitation**: Accept invitations to join teams
- **Duplicate prevention**: System prevents duplicate invitations

## GitHub Integration Enhancements

- **GitHub App installation flow**: Streamlined setup for GitHub integration
- **Additional GitHub tools**: Expanded capabilities for GitHub operations
- **Pull request events**: Workflow triggers for PR opened, ready_for_review, and closed events
- **Issue events**: Workflow triggers for issues opened and closed

## Pro Model Access

New **upgrade dialog** for Pro language models, making it easy to access premium AI capabilities with clear upgrade paths.

## Gemini Updates

- Added **Gemini 2.5 Flash Preview** (04-17) model
- Added **Gemini 2.5 Pro Preview** (03-25) model
