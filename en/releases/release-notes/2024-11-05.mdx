This update introduces the first workflow execution capabilities and adds GitHub authentication for seamless login.

## Workflow Execution

The first **workflow execution** feature is here! Workflows built with the editor can now be executed, marking a major milestone in Giselle's evolution from a design tool to a complete AI workflow automation platform.

- **Execute workflows**: Run your designed workflows and see results
- **Real-time feedback**: Monitor execution progress
- **Foundation for automation**: The beginning of automated AI workflows

## GitHub Login

New **GitHub Login** integration for quick and easy authentication. Sign in with your GitHub account to get started immediately.

## GitHub App Integration

Early **GitHub App integration** groundwork for connecting Giselle to your GitHub repositories and workflows.

## Model Configuration

- **Temperature and Top-P parameters**: Fine-tune AI model behavior with configurable parameters
- **Model provider support**: Foundation for multiple AI model providers
